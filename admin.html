📋 คำแนะนำสำหรับ Smart TV และหน้าจอขนาดใหญ่

  1. เปิด Browser บน Smart TV (Samsung Internet, LG WebOS Browser)
  2. เข้าไปที่ URL: https://vipart.netlify.app
  3. กดปุ่ม F11 หรือเข้า Full Screen Mode
  4. ใช้รีโมทหรือคีย์บอร์ด:
    • 1 = เปลี่ยนเป็นแนวตั้ง
    • 2 = เปลี่ยนเป็นแนวนอน
    • C = แสดง/ซ่อนตัวควบคุม
    • F = เข้า/ออกจากโหมดเต็มจอ
    • U = อัพเดทข้อมูล API ทันที
    • R = รีเฟรชหน้า
    • → = เปลี่ยนโฆษณาถัดไป
